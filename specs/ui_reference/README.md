# UI Reference 页面风格说明

本目录提供执法办案系统的参考页面风格（UI Ground Truth）。

## 使用说明
- 本样式仅用于页面效果对照与验收参考
- 不要求逐像素一致，但整体风格、层级、密度需保持一致或更优
- 可使用任意前端技术栈实现（Vue / React / 原生 HTML 等）

## 目录结构
- index.html    页面结构示例
- styles.css   样式定义
- assets/      静态资源（如 logo）

## 语义说明
该 UI Reference 不作为实现指令，不直接进入 BUILD 任务，仅用于 Planning / Review 阶段判断页面效果是否存在偏差。
